# Calculator Project

A beginner-friendly Python calculator that performs basic arithmetic operations.  
This project uses Python’s `match-case` statement for clear control flow and includes error handling with `try-except`.

## Features
- Addition, subtraction, multiplication, and division  
- Simple text-based interface  
- Handles invalid inputs gracefully  

## How to Run
1. Save the file as `main.py`.  
2. Run the program using:python main.py
3. Enter two numbers when prompted.  
4. Choose the operation (`+`, `-`, `*`, `/`).  
5. The result will be displayed instantly.  


## Learning Outcome
This project helped me practice:
- Taking user input with `input()`  
- Using `match-case` for branching logic  
- Exception handling with `try-except`  
- Performing basic arithmetic operations
